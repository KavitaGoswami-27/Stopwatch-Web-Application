
**⏱ Stopwatch Web Application**  
A modern and interactive **Stopwatch Web Application** built using **HTML, CSS, and JavaScript**. This project provides accurate time tracking with lap recording, statistics, dark mode, keyboard shortcuts, and a responsive user interface.
**🚀 Features**
- ▶️ Start Stopwatch
- ⏸️ Pause Stopwatch
- 🔄 Reset Stopwatch
- 🏁 Record Unlimited Lap Times
- ⏱ Display Time in **HH : MM : SS : MS**
- 📊 Total Lap Counter
- 🏆 Best Lap Detection
- 📈 Average Lap Time
- 📅 Live Date & Time
- 🌙 Dark / Light Mode
- ⌨️ Keyboard Shortcuts
- 📄 Export Lap Times as TXT File
- ✨ Animated UI & Hover Effects
- 📱 Responsive Design
- 🟢 Live Status Indicator (Running / Paused / Stopped)

**⌨️ Keyboard Shortcuts**

You can control the stopwatch using the following keyboard shortcuts for a faster and more convenient experience:

| **Shortcut Key** | **Function** |
|:----------------:|-------------|
| **Space** | ▶️ Start or ⏸️ Pause the stopwatch |
| **L** | 🏁 Record the current lap time |
| **R** | 🔄 Reset the stopwatch and clear all lap records |

> **Note:** The **Space** key works as a toggle—it starts the stopwatch if it's stopped and pauses it if it's running.
